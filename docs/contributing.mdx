## Open Source Spirit

Thank you for considering contributing to WCE!

By contributing, you're helping improve a robust tool designed for building WhatsApp chatbots and using WhatsApp Cloud APIs effortlessly.

---

<Warning>
  Before running, you must have your WhatsApp test credentials all set
</Warning>

> Checkout the getting started to [WhatsApp Cloud API](https://developers.facebook.com/docs/whatsapp/cloud-api/get-started/).

A good starting point for contributing is checking the corresponding engine repository contributing guide

**PYWCE** - [contributing guide](https://github.com/DonnC/pywce/blob/master/CONTRIBUTING.md)

**JAWCE** - [contributing guide](https://github.com/DonnC/jawce/blob/master/CONTRIBUTING.md)
